ログイン前のアクション
	join - 登録のためのメールを送信します
		現在有効な他のメールアドレスと同じメールアドレスが渡された場合はエラーになります。

	register - 登録します
		存在しないトークンの場合はエラーになります。
		成功したトークン以前のトークンの場合はエラーになります。
		有効期限を過ぎたトークンの場合はエラーになります。
		ログインページヘ遷移します。

	login - ログインします
		該当アクターページヘ遷移します。

	change_email_verify - メールアドレスを変更します
		存在しないトークンの場合はエラーになります。
		成功したトークン以前のトークンの場合はエラーになります。
		有効期限を過ぎたトークンの場合はエラーになります。
		ログインページヘ遷移します。

	forgot_password - パスワードリセットのためのメールを送信します

	reset_password - パスワードをリセットします
		存在しないトークンの場合はエラーになります。
		成功したトークン以前のトークンの場合はエラーになります。
		有効期限を過ぎたトークンの場合はエラーになります。
		ログインページヘ遷移します。


ログイン後のアクション
	unregister - 登録解除します
		ログアウトします。
		ゲストアクターページヘ遷移します。

	logout - ログアウトします
		ゲストアクターページヘ遷移します。

	change_email - メールアドレス変更のためのメールを送信します

	change_password - パスワードを変更します
		ログアウトします。
		ゲストアクターページヘ遷移します。
